# 🍰 Salgados & Cia - Sistema de Pedidos Online

## 📋 O que é esta aplicação?

O **Salgados & Cia** é um sistema web completo para pedidos online de uma confeitaria/padaria brasileira localizada em Palmas, PR. A aplicação permite que clientes naveguem pelo catálogo de produtos, adicionem itens ao carrinho e finalizem pedidos diretamente via WhatsApp.

## 🎯 Objetivo Principal

Facilitar o processo de pedidos para festas e eventos, oferecendo uma interface moderna e intuitiva que conecta clientes diretamente com o negócio através do WhatsApp, eliminando a necessidade de sistemas de pagamento complexos.

## 🛍️ Catálogo de Produtos

### Categorias Disponíveis:

1. **🥧 Tortas Assadas**
   - Torta Hot Dog
   - Torta de Frango com Requeijão  
   - Torta de Legumes
   - *Vendidas por unidade*

2. **🧁 Caseirinhos (Bolos Pequenos)**
   - Cenoura com Brigadeiro Gourmet
   - Charge (inspirado no chocolate)
   - Fubá com Erva Doce
   - *Vendidos por unidade*

3. **🍬 Doces**
   - Brigadeiro de Café
   - Brigadeiro de Limão
   - Brigadeiro de Paçoca
   - *Vendidos por cento (mínimo 25 unidades)*

4. **🍝 Lasanhas e Empadão**
   - Empadão (frango com requeijão ou palmito)
   - *Vendidos por kg (incrementos de 0,5kg)*

## ⚡ Funcionalidades Principais

### Para o Cliente:
- **Navegação Intuitiva**: Catálogo organizado por categorias
- **Busca Inteligente**: Encontre produtos rapidamente
- **Carrinho Persistente**: Seus itens ficam salvos entre sessões
- **Promoções Ativas**: Visualize ofertas especiais com badges
- **Pedido via WhatsApp**: Finalização simples e direta
- **Design Responsivo**: Funciona perfeitamente em celular, tablet e desktop

### Para o Negócio:
- **Gestão de Produtos**: Sistema completo de catálogo
- **Sistema de Promoções**: Controle de ofertas com datas
- **Multi-tenant**: Suporte para múltiplos estabelecimentos
- **Sem Complexidade**: Não requer sistema de pagamento online

## 🔧 Como Funciona

1. **Cliente acessa o site** → Visualiza o catálogo completo
2. **Navega pelos produtos** → Usa busca ou navega por categorias  
3. **Adiciona ao carrinho** → Escolhe quantidades respeitando mínimos
4. **Revisa o pedido** → Confere itens e valores
5. **Finaliza via WhatsApp** → Clica e é redirecionado com pedido formatado
6. **Comerciante recebe** → Pedido chega formatado no WhatsApp do negócio

## 💰 Sistema de Precificação

- **Por Unidade**: Tortas e bolos individuais
- **Por Cento**: Docinhos (mínimo 25, vendidos em lotes de 100)
- **Por Kg**: Empadão e lasanhas (incrementos de 0,5kg)
- **Promoções**: Sistema automático de preços promocionais com datas

## 🚀 Tecnologias Utilizadas

### Frontend:
- **React + TypeScript**: Interface moderna e tipada
- **Vite**: Build tool rápido e eficiente
- **Tailwind CSS**: Estilização responsiva
- **Local Storage**: Persistência do carrinho

### Backend/Database:
- **Supabase**: Banco PostgreSQL na nuvem
- **Multi-tenant**: Arquitetura para múltiplos clientes
- **SQL**: Estrutura robusta de dados
